# Proving

Prefer synthetic, reversible, least-consequential tests. A proving plan states
what could falsify the candidate, permitted conclusions, affected-party standing,
stop conditions, data boundary, recovery, and who owns the result. No repository
has a standing obligation to serve as proving ground.
