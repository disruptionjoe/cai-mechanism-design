# CMD-0001-P4 Disposition - Revise

P4 does not earn a bounded pass. The current typed register can preserve
source identity, provisional validation, possible contradiction, stop state,
and lower domain-owner routes, but it cannot type the source owner's mixed-fit
classification or the domain absorbers that make that classification honest.

A formally complete row could hide those two facts in prose and later be
misread as ordinary positive evidence. That is a paperwork-theater failure at
the translation boundary.

Required revision: create a minimal translation wrapper with typed
`source_owner_fit_status` and `absorber_or_counterevidence` fields, keep it
separate from source truth and the operational register, and rerun P4 against
the same frozen fixture.

This disposition does not accept another owner's schema, add a source claim,
prescribe a nuclear, tsunami, regulatory, health, emergency, legal, or policy
remedy, or authorize graduation, transfer, deployment, participation, field
testing, incentives, institutions, spin-out, destination obligations, or
external action.

## Resolution

`proving/CMD-0001-P4-typed-translation-wrapper.md` implemented this exact
revision. `proving/CMD-0001-P4-mixed-fixture-rerun-score.md` reran the same
frozen fixture and earned a bounded translation pass. The active disposition is
now `CMD-0001-P4-defer.md`; this revise record remains the preserved reason for
the wrapper.
