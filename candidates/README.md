# Candidates

No candidate is admitted at formation. Each future candidate needs a source
inquiry, theory of change, alternatives including no action, owner fit, affected
people, legitimacy and capture risks, evidence route, stops, correction route,
and intended disposition. Existing-owner fit defeats admission here.
